Complete guide to implementing AI-powered summaries on your website

What is ZaTwo?

ZaTwo is an AI-powered content summarization system that helps website owners provide concise, digestible summaries of their content. Instead of overwhelming visitors with lengthy SEO-optimized articles, you can offer instant summaries that keep users engaged.

Key Benefits

  • Automatic Summarization: AI generates summaries when users visit your pages
  • Zero Configuration: Just verify your domain and start using
  • Instant Results: Summaries generated in ~5 seconds on first request
  • SEO Friendly: Keep your original content while providing summaries

Getting Started

1Create an Account

Visit the registration page and create your account with your domain name.

2Verify Your Domain

To enable automatic summarization, you need to verify domain ownership:

  1. Go to the Domain Verification section in your dashboard
  2. Click "Request Verification" to get your unique TXT record
  3. Add the TXT record to your domain's DNS settings: 
    Type: TXT
Name: _domain-verification.yourdomain.com
Value: [your-unique-token]
  4. Click "Verify Domain" to complete the process

Important

Automatic summarization only works for verified domains. Without verification, you'll need to manually submit URLs for summarization.

3Implement the Summary Viewer

Add the following code to your website to display summaries:

HTML Implementation 

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <style>
    body {
      margin: 0;
      font-family: Arial;
    }
    .btn {
      padding: 15px 40px;
      background: #007bff;
      color: #fff;
      border: none;
      border-radius: 8px;
    }
    iframe {
      width: 100%;
      height: 100vh;
    }
  </style>
</head>
<body>
  <button class="btn" onclick="show()">View Summary</button>
  <iframe id="f"></iframe>
  <script>
    function show() {
      const u = new URLSearchParams(location.search)
        .get('url') || location.href;
      const p = new URL(u.startsWith('http') ? 
        u : 'https://' + u);
      const path = p.hostname + p.pathname + p.search + p.hash;
      const base = 'https://c.za2.net';
      document.getElementById('f').src = 
        base + '/' + path.replace(/^\/+/, '');
    }
  </script>
</body>
</html>

How It Works

  1. The script extracts the current page URL (or from ?url= parameter)
  2. Constructs a summary URL: base-url/domain/path
  3. Loads the summary in an iframe when user clicks the button

Example Usage

If your page is: https://example.com/blog/article

Summary URL becomes: https://c.za2.net/example.com/blog/article

Automatic Summarization

Once your domain is verified, the system works automatically:

First Visit

When a user requests a summary for the first time, the system crawls and summarizes the page in ~5 seconds.

Subsequent Visits

Pre-generated summaries load instantly for returning visitors.

No Manual Work

No need to manually submit URLs - everything happens automatically!

How Automatic Summarization Works

  1. User clicks "View Summary" button on your website
  2. System checks if summary exists for that URL
  3. If not found and domain is verified:
    • Page is automatically crawled
    • AI generates summary (~5 seconds)
    • Summary is displayed and cached
  4. Future requests load the cached summary instantly

Manual URL Submission

You can also manually submit URLs for summarization:

  1. Go to URL Management in your dashboard
  2. Enter the full URL you want to summarize
  3. Click "Submit"
  4. Wait for AI to process (usually 5-10 seconds)
  5. View your summary in the history section

Subdomain Support

After verifying your main domain, you can add subdomains:

  1. Go to Domain Verification
  2. Scroll to "Subdomains" section
  3. Add subdomains (e.g., blog.example.com)
  4. Subdomains inherit the main domain's verification

Menu Customization

Customize the menu items shown in summary pages:

  1. Navigate to Menu Customization
  2. Add up to 10 custom menu items
  3. Each item can have:
    • Button text (max 60 characters)
    • Target URL (where button links to)
  4. Drag to reorder items
  5. Save changes

Integration Methods

Method 1: Iframe Embed (Recommended)

Use the HTML code provided above. This is the simplest and most reliable method.

Method 2: Direct Link

Link users directly to the summary URL:

<a href="https://c.za2.net/example.com/page" 
   target="_blank">
  View AI Summary
</a>

Method 3: Custom Integration

Build your own viewer using the summary URL format:

https://c.za2.net/{domain}/{path}

Troubleshooting

Common Issues

  • Summary not generating? Ensure your domain is verified
  • 404 errors? Check that the page URL is accessible and doesn't redirect
  • Slow processing? First-time summaries take ~5 seconds; this is normal
  • DNS verification failing? Wait 5-10 minutes for DNS propagation

Usage Limits

Current Limits

  • Daily URL submissions: 10 manual submissions per day
  • Automatic summaries: Unlimited (for verified domains)
  • Menu items: Up to 10 custom items
  • Subdomains: Unlimited
